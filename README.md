# 🚀 CodeSnap

A blazing fast terminal code snippet manager for developers.

## Features

- ⚡ Fast and lightweight
- 🔍 Fuzzy search
- 📋 Copy to clipboard
- ▶️ Execute snippets
- 🏷️ Tagging system
- 🎨 Syntax highlighting
- 📤 Import/Export

## Installation

```bash
pip install codesnap-cli
```

## Quick Start

```bash
# Add a snippet
codesnap add --title "Hello World" --code "print('Hello, World!')" --language python

# Search snippets
codesnap search "hello"

# List all snippets
codesnap list

# Show a snippet
codesnap show <id>

# Copy to clipboard
codesnap copy <id>
```
